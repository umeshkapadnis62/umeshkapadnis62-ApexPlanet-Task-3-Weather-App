# 🌦️ ApexPlanet Internship - Task 3
## Weather App using API Integration

This project is a Weather Web Application developed using HTML, CSS, and JavaScript with API integration.

---

## 🚀 Features

- Search weather by city name
- Displays Temperature, Weather Condition, Humidity, and Wind Speed
- Real-time data using OpenWeather API
- User-friendly interface
- Error handling for invalid city names

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Fetch API, DOM Manipulation)
- OpenWeather API

---

## 📌 How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Enter city name and click search

---
